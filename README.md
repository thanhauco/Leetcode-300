# Leetcode 500

A comprehensive collection of **500 Leetcode questions** from the past 3 years, organized by pattern and topic.

## Structure

The repository is organized by category:
- Arrays & Hashing
- Two Pointers
- Sliding Window
- Stack
- Binary Search
- Linked List
- Trees
- Tries
- Heap & Priority Queue
- Backtracking
- Graphs
- Advanced Graphs
- 1-D DP
- 2-D DP
- Greedy
- Intervals
- Math & Geometry
- Bit Manipulation

## Content

Each problem directory contains:
- `README.md`: 
    - Problem statement
    - **ASCII Art Visualization**
    - Mermaid Flowchart
    - Analytics (Time/Space complexity)
    - Hints
- `solution.py`: Python solution with detailed comments explaining the logic.

## Progress

- **Total Problems**: 500
- **Timeline**: May 2023 - October 2024
- **Focus**: Heavy emphasis on Medium & Hard problems in the later stages.
